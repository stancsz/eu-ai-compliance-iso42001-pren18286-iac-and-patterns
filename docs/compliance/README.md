# Compliance Templates

Templates and playbooks that implement ISO/IEC 42001 management system requirements and EU AI Act high-risk obligations. Instantiate these documents per solution and link to evidence repositories (e.g., Confluence, SharePoint, Jira).

## Available Templates

- `risk_management_plan.md`
- `data_gov_playbook.md`
- `technical_documentation_index.md`
- `logging_monitoring_plan.md`
- `human_oversight_playbook.md`
- `pmm_plan.md`
- `internal_audit_checklist.md`
